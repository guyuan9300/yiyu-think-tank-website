- 已打通：Feishu 通道支持通过手机号解析并给“尚未私聊过的用户”发 DM。
  - 用法（方案A）：message tool 的 target 写成 `user:mobile:<手机号>`（例如 `user:mobile:18027370767`）。
  - 前置：飞书开放平台开通 contact-v3 的手机号换 ID 权限；OpenClaw 配置启用：
    - `channels.feishu.mobileLookup.enabled=true`
    - 可选：`cacheTtlMs=86400000`、`includeResigned=false`
  - 实现：插件内部调用 `client.contact.user.batchGetId` 把 mobile -> open_id，再走原 `im.message.create`。
  - 注意：仅 SIGUSR1 热重载不一定会加载到插件代码变更；需要完整重启 OpenClaw 进程后生效。
